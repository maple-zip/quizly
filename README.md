# Quizly - Nền tảng ôn tập trực tuyến

Quizly là là một nền tảng nhỏ tạo và làm bài ôn tập trực tuyến miễn phí, mã nguồn mở dựa trên [Supabase](https://supabase.com/)

## Tính năng chính

### Editor (Soạn đề)
- Tạo câu hỏi trắc nghiệm ABCD, Đúng/Sai
- Upload hình ảnh, audio, video
- Tùy chỉnh thời gian, xáo trộn câu hỏi, đáp án.
- Xuất file ZIP (để dùng sau) hoặc đăng lên cloud

### Review (Làm bài)
- Giao diện responsive trên điện thoại/desktop
- Bộ đếm giờ, cảnh báo hết thời gian
- Chấm điểm tự động, hiển thị kết quả chi tiết
- Hỗ trợ playback media trực tiếp

### Cloud Storage
- Backend chạy trên [Supabase](https://supabase.com/)
- Tự động xóa sau khoảng 7 ngày
- Chia sẻ bài ôn tập bằng link đơn giản
